# Syntecxhub_AI-Mock-Interview-Chatbot
A Python-based conversational interview bot that simulates technical and HR interviews.The bot evaluates answers using keyword-based matching and generates performance reports.


## Features
-Rule-Based AI Chatbot
- Technical interview simulation
- HR round simulation
- Keyword-based answer evaluation
- Performance scoring
- Interview history storage
- 15 Random Technical Questions
- 7 Random HR Questions
- Keyword-based Evaluation
- Partial Marking System
- Synonym Support
- Interview ID & Timestamp
- Final Result: Selected / Rejected
- Detailed Interview Report


## How it Works
-The bot asks 15 random technical questions and evaluates answers based on keywords and synonyms, giving partial marks for incomplete answers.
-If the candidate scores ≥ 60%, they proceed to 7 random HR questions, also scored with keyword-based evaluation.
-A final report is generated with total scores, detailed breakdown, and a Selected / Rejected result

-Assessment Tool
-Learning & Self-Improvement

- Rule-based chatbot logic
